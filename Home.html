Welcome to my how-to guide on building a simple one person blackjack game using the deckofcards API



In this guide, I will cover the following topics:

  • A brief introduction to the API and how to get and receive information
  • How to set up a simple user interface to play the game
  • Creating functions using AJAX (Asynchronous JavaScript And XML) requests
  • Combining it all together to get a working game
  • Lastly, I will show you what a complete game looks like


To get the most out of this guide you should have a basic understanding of html, javascript and GET requests and DOM. The list below has some useful links for brushing up if you need a refresher.
