User Interface using simple HTML



 <div id="playerHand"><h4>Player's Cards:</h4>
    Value: <span id="pDeck"></span><br>
    <img src="" alt="Card 1"
         style="width:104px;height:142px;" name="pCards" id="pCard1">
    <img src="" alt="Card 2"
         style="width:104px;height:142px;" name="pCards" id="pCard2">
</div>

<div id="dealerHand"><h4>Dealer's Cards:</h4>
    Value: <span id="dDeck"></span><br>
    <img src="" alt="Card 1"
         style="width:104px;height:142px;" name="cards" id="dCard1">
    <img src="" alt="Card 2"
         style="width:104px;height:142px;" name="cards" id="dCard2">
</div>

<form>
    <fieldset>
        <legend>Blackjack</legend>
        <button type="button" id="newHand">New Hand</button>
        <button type="button" id="hit">HIT!</button>
        <button type="button" id="stay">STAY!</button>
</fieldset>
</form>
<div id="score">
    Rounds:<span id="rounds" style="padding-right:20px"></span>
    Player Wins:<span id="pWins" style="padding-right: 20px"></span>
    Dealer Wins:<span id="dWins" style="padding-right: 20px"></span>
</div>

This is the only code we are using in our html file. As you can see it starts a h4title for the player's hand followed by a span that displays the value of the cards. Then it has two img holders for the cards.
The second block is the same showing the dealer's hand.

In the next section well discuss how we add more images using DOM to display addtional cards that are drawn. Technically speaking all of the cards could have been displayed using DOM but since we know every hand will have atleast two cards, I felt it was best to display the spots where those cards would appear.

The next block of code contains our buttons that the user will use to play the game. In the combining it together section, I show how to make these buttons function properly.

The last div here will display the score for the player and the dealer, and will keep track of the number of rounds. If you scroll down you can see what this page would look like.
