At this point we are almost done. We have set up a static html page and written a whole bunch of code to run the game. The last thing to do is connect it all together.

We'll start by making sure we link our js file to the html page. We do this by adding a line of code like this:
<script src="blackJack.js"></script> just above the </body> line in our html file.

Now that our js file is linked, let's make sure the buttons actually do some thing. Let's head back into our js file and add the follwing code just below our variables list. 

document.getElementById('newHand').addEventListener('click', newHand );
document.getElementById('hit').addEventListener('click', hit );
document.getElementById('stay').addEventListener('click', stay );
What this is doing is getting the elements from the html page (in this case its the buttons) and adding and event listener addEventListener, that will call the function to the right when the action click happens.
Now when ever a user clicks one of the buttons, the newHand, hit, or stay functions will be called.

The only thing left to do is add the last function mentioned on the jsFunctions page to display the score. 

function addScore(who){
    who.win++;
    round++;
    document.getElementById('pWins').textContent = pDeck.win;
    document.getElementById('dWins').textContent = dDeck.win;
    document.getElementById('rounds').textContent = round;
}
All this code does is take as a parameter, the object that won (pDeck or dDeck) and adds 1 to their 'win' property. It then increases the rounds variable by 1 and displays both scores and the number of rounds to the page.
